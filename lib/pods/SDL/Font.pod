=head1 NAME

SDL::Font - a SDL perl extension

=head1 SYNOPSIS

  $font = new Font "Font.png";
  $font->use();
	
=head1 DESCRIPTION

L<SDL::Font> provides an interface to loading and using SFont style 
fonts with L<SDL::Surface> objects.  

=head1 METHOD

=head2 use(font)

Use the SDL::Font  given as first parameter.
